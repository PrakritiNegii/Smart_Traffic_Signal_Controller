## Traffic Signal Management System (v1)

This project implements a junction-level adaptive traffic signal controller.

### Features
- Vehicle detection using YOLOv5
- Junction-level modeling with 4 realistic signal phases
- Rule-based phase selection using vehicle counts
- Anti-starvation logic to prevent phase lock
- Tkinter-based UI for junction decision visualization

### Current Version (v1)
- Static image-based junction snapshots
- Manual cycle triggering
- Rule-based control logic

### Planned Improvements
- Automatic phase cycling with real-time animation
- Traffic light visualization (red/yellow/green)
- ML-based traffic prediction
- Waiting-time based phase prioritization
