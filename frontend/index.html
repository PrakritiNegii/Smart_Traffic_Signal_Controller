System Architecture

How SignalFlow detects vehicles and controls junction signals

SignalFlow is an adaptive traffic signal controller. It uses YOLOv5 to count vehicles from lane snapshots, then applies rule-based logic to pick the green phase and calculate signal timing — with anti-starvation so one direction never gets stuck on red forever.

Architecture Diagram

How It Works

  1. 1
    Upload 4 lane images

    North, South, East, West snapshots from the junction camera feed.

  2. 2
    YOLO vehicle detection

    Each image is scanned for cars, buses, trucks, and motorbikes.

  3. 3
    Count vehicles per lane

    Builds a traffic load map: N, S, E, W vehicle counts.

  4. 4
    Phase selection

    Picks P1–P4 based on highest load, with anti-starvation rotation.

  5. 5
    Adaptive green timing

    Trucks/buses get longer green; result clamped between min and max.

  6. 6
    Signal visualization

    Dashboard shows red / yellow / green and auto-cycles through phases.
